species_abundance_SPEI_L1
name	definition
year	Year in which data was collected
site	The site from which the data originates.
species	Taxon name as it occurs in the original data. Harmonized species names accross datasets are in the column species harmonized
dataset	The dataset from which the data originates and usually corresponds with the experiment from which the data was collected. This was only entered for Cedar Creek data.
plot	plot number/id within experiment/site
higher_order_organization	Grouping variable for the next higher organization level than plot. For example Site_Experiment_Block, if the data orginiates from an experiment at a site that is organized in blocks. The variable structure depends on the experimental design from which the data point originates.
uniqueid	Unique plot id accross experiments and sites.
abundance	Species abundance as measured in its original unit (biomass, %cover at various spatial scales)
relative_abundance	the relative abundance of the species in any year and plot calculated based on the measured abundance
original_measurement_unit	Indicates at which unit the value in the abundance column was estimated
cover_method	additional method how relative abundance was estimated
species_harmonized	The binomial species name or Genus sp., harmonized accross datasets.
identification_level	The taxonomic detail to which a species was identified.
broad_category	information on whether the value is at the species level or higher, or if it is unidentified or abiotic
spei12	annual Standardized Precipitation Evapotranspiration Index for August from SPEIbase
spei3	3-month Standardized Precipitation Evapotranspiration Index for August from SPEIbase
spei6	6-month Standardized Precipitation Evapotranspiration Index for August from SPEIbase
spei9	9-month Standardized Precipitation Evapotranspiration Index for August from SPEIbase
spei3_category	Categorical values representing climatic conditions based on SPEI-3: extreme drought, extreme wetness, moderate drought, moderate wetness, and normal.
spei6_category	Categorical values representing climatic conditions based on SPEI-6: extreme drought, extreme wetness, moderate drought, moderate wetness, and normal.
spei9_category	Categorical values representing climatic conditions based on SPEI-9: extreme drought, extreme wetness, moderate drought, moderate wetness, and normal.
spei12_category	Categorical values representing climatic conditions based on SPEI-12: extreme drought, extreme wetness, moderate drought, moderate wetness, and normal.

ece_resist_resil_spei9_norm_L2
name	definition
ex_year	extreme climate event year
site	Long-Term Ecological research site
uniqueid	unique identifier for a plot
resistance	mean normal year biomass divided by the absolute value of event year biomass minus mean normal year biomass
resilience	the absolute value of event year biomass minus mean normal year biomass divided by next year's biomass minus mean normal year biomass

metadata_L1
name	definition
year	Year in which data was collected
site	The site from which the data originates.
dataset	The dataset from which the data originates and usually corresponds with the experiment from which the data was collected. This was only entered for Cedar Creek data.
plot	plot number/id within experiment/site
higher_order_organization	Grouping variable for the next higher organization level than plot. For example Site_Experiment_Block, if the data orginiates from an experiment at a site that is organized in blocks. The variable structure depends on the experimental design from which the data point originates.
uniqueid	Unique plot id accross experiments and sites.
growprecip	Sum of precipitation per year at site level within a growing season period defined by a range of Julian day of year dates from this paper: https://doi.org/10.1111/j.1600-0706.2012.20655.x
treatment	were plots control or treated (e.g., fertilization, tilled, etc.)
nutrients_added	Type of nutrients added to plot.
nitrogen_amount	Amount of nitrogen added to plot.
disturbance	Note of whether plots were experimentally disturbed (e.g., annual tilling)
grazing	Whether or not a plot was grazed or not.
fire_frequency	How often plots were experimentally burned (1 = annually).
time_since_fire	the number of years since the last prescribed burn
source	Where data came from. This can be the data code (PAB012 for Konza, Table 250 for KBS, etc). Can be a list of sources. This aids in finding dataset on LTER website if issues/questions arise.
treatment_comment	Warnings if treatments were changed. Treatments can change over time. E.g. stop of fertilization, start of burning. So the same plot can have different treatment variables in different time points. OR the plots are treated as different plots with different treatments once the treatments change. OR the original treatment is the treatment that we look at.
diversity_manipulated	was this an experiment that manipulated diversity or were grasslands naturally assembled
meantemp	Annual mean temperature at site level.
annualprecip	Annual precipitation at site level.
phosphorus_amount	Amount of phosphorus added to a plot.
